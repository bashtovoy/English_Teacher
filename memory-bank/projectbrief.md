# Project Brief — English_Teacher (VOT)

## Version
1.13.1

## Description
Browser extension for video voice-over translation with subtitle support, audio downloading, and language learning features. Supports YouTube and other video platforms with smart layout, Yandex translation, and multi-language UI (50+ languages).

## Repository
https://github.com/bashtovoy/English_Teacher.git

## Core Requirements
- Add Yandex Browser-style voice-over translation to other browsers (Chrome, Firefox, etc.)
- Support subtitle display with smart layout algorithm
- Enable audio downloading from video platforms
- Provide language learning features with phrase alignment

## Goals
- Make video content accessible across languages through voice-over translation
- Support multiple video platforms (YouTube + others)
- Provide a seamless user experience with minimal configuration

## Scope
- Browser extension (Chrome/Firefox) and userscript builds
- Integration with Yandex translation API
- Custom subtitle rendering engine
- Audio download functionality
