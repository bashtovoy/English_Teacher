# Product Context — English_Teacher (VOT)

## Why This Project Exists

### Problems It Solves
- Users want voice-over translation in browsers that don't have native Yandex Browser integration
- Video content is language-barriered; translation breaks down accessibility
- Existing solutions lack subtitle customization and audio download features

### User Experience Goals
- One-click translation on any supported video platform
- Smart subtitle layout with language learning features
- Ability to download translated audio for offline use
- Multi-language UI support (50+ languages)
